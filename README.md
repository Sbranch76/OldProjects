# OldProjects
这里存放了我大一时期的课程项目与练习代码，主要使用 **C 语言** 和 **Java** 完成。

## 项目列表
### 🎮 C语言六子棋游戏（大一C语言大作业）
- **开发环境**：Visual Studio 2022 + EasyX图形库 + Win11
- **核心功能**：双人/人机对战（3级AI难度）、棋局计时/胜负判定、棋谱保存/回放、游戏存档/读档、背景音乐/音量调节、战绩统计
- **实现亮点**：图形化界面绘制，自定义AI落子算法，文件操作实现数据持久化，双缓冲优化界面流畅度

### 📁 Java文件检索与去重工具（JAVA）
- **开发环境**：IDEA 2021.3.1 + JDK11 + Win11
- **核心功能**：目录递归扫描、文件信息统计、MD5哈希检测重复文件、TXT报告导出、重复文件安全删除
- **实现亮点**：模块化功能设计，8KB缓冲区优化大文件处理，全面异常处理（权限/路径/非法输入），熟练运用集合框架与IO流

> 注：这些项目是我编程入门阶段的实践，部分功能由 AI 辅助完成。
